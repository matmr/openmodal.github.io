About OpenModal

We are developing a Python based environment for experimental modal analysis. The first public presentation will be at the www.icoev.org conference (Ljubljana, Sep 2015). The code is not yet public; however, we plan to go public early 2016.
OpenModal was started by PhD graduates of www.ladisk.si
